Integrated MC 是地牢建筑统合（IDAS）系列模组的官方整合包，专注于打造最出色的探索体验。 通过任务、配方、背景故事、战利品以及各种精心设计的建筑，本整合包将每一个模组巧妙融合在一起，不再让模组显得像单独的支线任务， 而是呈现出一款浑然一体的游戏。
